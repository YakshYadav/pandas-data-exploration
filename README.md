# pandas-data-exploration
# Pandas Data Exploration and Cleaning

This is my first data analysis project using Python and Pandas.

# What I did

I loaded a shopping dataset and performed basic data cleaning and exploration.

# Steps

1. Loaded the CSV file into a Pandas DataFrame
2. Explored the data using head(), shape, and dtypes
3. Found and filled missing values with the median
4. Filtered rows and selected specific columns
5. Removed duplicate rows
6. Created a new column: total_amount = price × quantity
7. Saved the cleaned data as a new CSV file

# Files

- Yaksh WEEK1.ipynb - main notebook 
- Yaksh Cleaned CSV.csv - cleaned output

- How to load and explore a dataset
